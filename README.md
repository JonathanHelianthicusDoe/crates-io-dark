# crates-io-dark

A dark theme for [crates.io](https://crates.io/).

## Install

Install [Stylus](https://addons.mozilla.org/en-US/firefox/addon/styl-us/) and
then use Stylus to install [the crates-io-dark
usercss](./crates-io-dark.user.css).
